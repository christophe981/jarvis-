# Outils

Configurations, templates, dashboards, fichiers de travail réutilisables.

Exemples : tableau de bord Notion, template de contrat OBM, kit de bienvenue client, modèle de proposition commerciale, fichier de suivi.

Convention : `AAAA-MM_NOM-OUTIL_type` (ex: `2026-06_tableau-bord-obm_notion`)
